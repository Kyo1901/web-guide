Process

스케치 4단계

손 스케치든 디지털 도구든, 아래 순서를 따르면 됩니다.

01
큰 구역 먼저
Header / Main / Footer 세 영역을 큰 사각형으로 나눕니다. 세부 내용은 나중에 채워요.
02
콘텐츠 배치
각 영역 안에 들어갈 요소(이미지, 텍스트, 버튼)를 박스로 표시합니다. 내용은 선만 그어요.
03
레이블 달기
각 박스가 무엇인지 이름을 씁니다. nav, hero, card, footer 등 HTML 태그 이름을 써도 좋아요.
04
모바일도 함께
PC 스케치 옆에 모바일 버전도 작게 그립니다. 2단 → 1단으로 바뀌는 부분을 확인해요.
Elements

페이지 구조 요소

모든 페이지는 아래 4가지 영역으로 구성됩니다.

🔝
Header
로고 + 내비게이션 메뉴가 들어가는 영역. 모든 페이지에 공통으로 들어갑니다.
🖼️
Hero / Banner
페이지 상단의 큰 이미지나 메시지 영역. 방문자가 처음 보는 핵심 화면입니다.
📦
Main Content
실제 콘텐츠가 들어가는 본문 영역. 카드, 텍스트, 이미지 등이 배치됩니다.
🔚
Footer
연락처, 저작권, 링크 등 부가 정보가 들어가는 하단 영역입니다.

Rules

와이어프레임 그리기 규칙

와이어프레임은 "구조"를 보여주는 것이므로, 예쁘게 그릴 필요가 없어요.

✓ 해야 할 것
  • □ 이미지 자리는 X 표시 또는 사선으로 채우기
  • □ 텍스트 자리는 수평선(——)으로 표시
  • □ 각 박스에 역할 이름 쓰기 (nav, hero, card...)
  • □ PC / 모바일 버전 따로 그리기
  • □ 가로 세로 비율 대략 맞추기
✗ 하지 말 것
  • □ 실제 색상 채우기 (색은 디자인 단계에서)
  • □ 실제 폰트 / 텍스트 내용 쓰기
  • □ 픽셀 단위로 정확하게 그리기
  • □ 완성도에 집착하기 (러프하게 OK)
